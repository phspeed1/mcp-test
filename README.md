# Todo 앱

할 일을 기록하고 관리하는 간단한 웹 애플리케이션입니다.

## 기능

- 할 일 추가
- 할 일 완료 체크
- 할 일 삭제
- 로컬 스토리지를 이용한 데이터 유지

## 기술 스택

- React
- HTML/CSS
- JavaScript
- LocalStorage

## 실행 방법

1. 저장소 복제
```
git clone https://github.com/phspeed1/mcp-test.git
cd mcp-test/todo-app
```

2. 의존성 설치
```
npm install
```

3. 개발 서버 실행
```
npm start
```

4. 브라우저에서 확인
```
http://localhost:3000
```
